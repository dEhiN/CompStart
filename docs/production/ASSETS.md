# CompStart/prodenv/assets

Contains all assets related to, or for use in, releases. This includes scripts, instruction files, etc.

## Directories

- _cs-installer_ - Contains two installer scripts - _install.ps1_ and _install.bat_.
- _release-assets_ - Contains master copies of the _instructions.txt_ and _release_notes.md_ files.

## Instructions

All four files are to be included in each release. The two files in the _release-assets_ folder are considered master copies. When these master copies are updated, previous release folders should keep the copies that were created at the time of release. Only future releases should use the updated versions of these two files.

For installation instructions, please refer to the _instructions.txt_ master file.
