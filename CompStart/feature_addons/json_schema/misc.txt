You can use the anyOf keyword to assert that your items can be one thing or another. anyOf is boolean OR (||) operation on schemas.
{
  "type": "array",
  "items": {
    "anyOf": [
      { "type": "string" },
      {
        "type": "array",
        "items": { "type": "string" }
      }
    ]
  }
}
