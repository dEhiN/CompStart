Note: Because CompStart is primarily meant as an automation tool, everything is done using scripts and is text-based. This means all interactions with the tool rely only on text and simple menu prompts with set choices.


How to install CompStart:

There are two installer scripts - install.bat and install.ps1. To install CompStart, double click on and run install.bat. That will run install.ps1. You should then see a text-based window appear. The installer will set up CompStart and will print to the window what it is doing so you can follow along.

CompStart is installed to the location C:\Users\<your username>\AppData\Local. For example, if your Windows profile or username is JohnDoe, then you will find the installation at C:\Users\JohnDoe\AppData\Local\CompStart. A shortcut to the CompStart.bat file is also created and placed in the Startup folder of your Start Menu.

This ensures that when you first log in, CompStart is run. The shortcut that's created will run CompStart.bat from the installed CompStart folder. This, in turn, will run CompStart.ps1, the main script that does all the magic!


How to add, remove, or change the programs that run on startup via CompStart:

Starting with release 0.1-beta, the tool CompStart.exe was added to assist with modifying the programs that the main CompStart script runs on login. The main script uses a configuration file that stores information about these programs. This information is referred to by the tool as startup data - the data that CompStart uses to know which programs to start up.

When you run CompStart.exe, a text-based window will appear with a main menu of choices. The main menu leads to different sub menus that appear based on what choices you make. The main menu and the sub menus allow you to view the current startup data


Note: The default startup data that comes pre-added opens Windows Calculator, Chrome, and Windows Notepad. For Chrome, the sites Google, Facebook, and X (formerly known as Twitter) are opened in a new window. This can all be changed using the instructions for "How to modify the startup data file".
